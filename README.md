# 🛡️ Report-Check.ai 

**Report-Check.ai** aik advanced web-based tool hai jo specifically Pentesting Reports ko analyze aur grade karne ke liye banaya gaya hai. Yeh tool teachers aur security professionals ki madad karta hai taake wo reports ko manually check karne ke bajaye AI ki madad se seconds mein evaluate kar saken.

## 🚀 Key Features

- **Automated Scoring:** Report ki quality aur structure ki bunyaad par automatically score (0-100%) calculate karta hai.
- **Vulnerability Recognition:** Report mein mojud critical bugs (jaise SQLi, XSS, RCE) ko pehchanta hai.
- **Structure Analysis:** Check karta hai ke zaruri sections (Executive Summary, POC, Remediation) mojud hain ya nahi.
- **Teacher's Feedback:** Har report ke liye behtri ke mashwary (improvements) suggest karta hai.
- **Futuristic UI:** Glassmorphism aur Dark Mode design jo user-experience ko behtreen banata hai.

## 🛠️ Tech Stack

- **Frontend & Backend:** [Streamlit](https://streamlit.io/) (Python Framework)
- **Data Processing:** Pandas & Regular Expressions (Regex)
- **Styling:** Custom CSS with Cyberpunk aesthetics

## 📂 Project Structure

- `app.py`: Main application logic aur UI code.
- `requirements.txt`: Zaruri libraries ki list (Streamlit, Pandas).
- `uploads/`: Temporary storage for report analysis.

## 🚦 How to Use

1. **Upload:** Apni pentesting report (Text format) ko upload button ke zariye select karen.
2. **Analyze:** Tool automatically content ko scan karega.
3. **Review:** Score, detected vulnerabilities, aur missing sections ko dashboard par mulahiza karen.
4. **Improve:** Diye gaye feedback ki roshni mein report ko behtar banayen.
